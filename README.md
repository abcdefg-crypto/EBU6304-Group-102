# EBU6304 Software Engineering Group Project
## International School TA Recruitment System

### Project Introduction
This project is a teaching assistant recruitment management system designed for BUPT International School. As part of the EBU6304 Software Engineering course, this system is developed using Java, Servlet/JSP and follows Agile development principles.

The system replaces traditional paper and Excel-based management with a digital platform, providing a complete online recruitment process for three types of users: TA applicants, Module Organisers (MO), and system administrators. It supports functions including registration, profile management, job posting, application, review, selection and workload statistics.

In line with course requirements, the system does not use any database. All data is stored and managed in plain text files such as TXT, JSON or XML.

### Tasks to Complete
#### 1. Applicant (TA) Tasks
- Log in to the system
- Create personal profile
- Edit personal profile
- Upload PDF resume
- View available TA positions
- Search for TA positions
- View position details
- Apply for a TA position
- Check application status
- Withdraw an application
- Receive application success notifications
- View missing skills for a position

#### 2. Module Organiser (MO) Tasks
- Publish TA job postings
- Edit job posting information
- View applicants for a position
- Review applicant resumes
- Approve/hire applicants
- Reject applicants
- Close job postings

#### 3. Admin Tasks
- View TA workload
- View all application records
- Generate workload reports
- Get workload balancing suggestions

#### 4. System Tasks
- Identify user roles after login
- Implement role-based access control
- Persist data using plain text files
- Provide unified error prompts
- Implement skill matching and recommendation

#### 5. Project Delivery Tasks
- Complete system testing and bug fixing
- Write full project documentation
- Prepare final submission and demo
